O que são TAGS?

(Voltar)

TAGS: no HTML são comandos que servem para formatar um texto ou palavra e frase, as tags sempre começam com o sinal de maior que (<(TAG)) e terminam com menor que (/TAG>), por exemplo:

  • (< p > )(< / p >), essa tag é usada para iniciar e terminar um parágrafo (w3schools)
  • (< strong > )(< / strong >) essa tag é usada para deixar a frase em negrito(w3schools)

Como é uma estrutura básica de um HTML?

(Voltar)

ESTRUTURA DO HTML: A estrutura basica do HTML é formada por 3 tags:

  1. < HTML> Ultilizado para definir o inicio e fim do HTML < / HTML>
  2. < HEAD> Onde contém informações, como o título da página e links para estilos e scripts< / HEAD>
  3. < BODY> Contém o conteúdo do documento < / BODY> /li>
